Xyfer

Xyfer is an API gateway with the goal of permeating the finance industry by revolutionizing the way banks perform overseas transfers. The API aims to reduce infrastructure costs, consumer transaction overhead, and also improve the speed at which these transactions happen. On top of that, it looks to uphold a consistent and scalable service with the use of the blockchain and through a decentralized network of adopting banks.

Tech stack:
- MongoDB
- HTML, CSS, Javascript
- NodeJS
- Solidity
- Misc. APIs

Checkscii

Checkscii is an experimental project that I developed in my free time. The idea was to algorithmically generate art on the Ethereum mainnet as ERC721 tokens. The algorithm employs two kinds of smart contract --- a front facing token contract and a descriptor contract. The art is stored on-chain, and does not employ any third-party remote storage. I provided the ability to mint these tokens by deploying a simple website using Vercel, created with some simple React components.

Tech stack:
- Hardhat
- Solidity
- React
